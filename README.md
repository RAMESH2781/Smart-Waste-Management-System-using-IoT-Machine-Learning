# Smart IoT Enabled Waste Management System

## Overview
This project is a complete smart waste management solution that combines
IoT sensors, Machine Learning models and a professional MERN stack dashboard.

## Features
- Wet / Dry waste detection
- Real-time sensor monitoring
- Methane and temperature tracking
- ML based risk and priority prediction
- Route optimization for waste collection
- Live interactive dashboard

## Hardware Used
ESP32, Ultrasonic Sensor, Soil Moisture Sensor, MQ4 Methane Sensor,
DHT11 Temperature Sensor, Neo-6M GPS Module, Battery, Jumper Wires

## Software & Tools
Arduino IDE, ThingSpeak, Python, Flask/FastAPI, Machine Learning,
React (MERN Stack), REST APIs

## Machine Learning Models
1. K-Means Clustering for priority classification
2. Random Forest for fill level prediction

## Data Flow
Sensors → ESP32 → ThingSpeak → API → ML Models → Backend → Dashboard

## Future Scope
Image based waste segregation, computer vision, mobile app,
advanced route optimization and large scale deployment.
